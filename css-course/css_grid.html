Grid

Grid patří také mezi novější vlastnosti v CSS 3, které jsou podporavány prohlížeči. Opět nám umožňuje práci s rozložením jednotlivých prvků. Používá se na zobrazení více řádků a sloupců. Proto se také grid někdy označuje, jako moderní tabulky.

Grid je velmi podobný, jako flex, pouze se od něj lehce liší.

Jak grid nastavit?

Grid nastavujeme pomocí vlastnosti display a jako hodnotu používáme grid. Nesmíme, ale zapomenout nastavit počet řádků a sloupců, které chceme, aby náš prvek měl. Počet sloupců označujeme vlastností grid-template-columns a počet řádků označujeme vlastností grid-template-rows.

Jednotky u sloupců a řádků

Řádky a sloupce můžeme nastavit absolutní jednotkou například v pixelech. Ale grid nám nabízí speciální jednotku, která se označuje fr. Tato jednotka reprezentuje jakousi část/zlomek velikosti našeho prvku.

Zarovnání prvků

Prvky můžeme zarovnávat na hlavní (X) a vedlější ose (Y). Pro hlavní osu (X) se používá vlastnost justify-items a může nabývat stejných hodnot, jako zarovnání na hlavní ose u flexu. Pro vedlejší osu (Y) se používá vlastnost align-items a může nabývat stejných hodnot, jako zarovnání na vedlejší ose u flexu.

Mezery mezi prvky

Mezery mezi prvky se vytváří stejně, jako u flexu. Používáme vlastnost gap.
