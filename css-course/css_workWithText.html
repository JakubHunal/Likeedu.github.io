Práce s textem

Pro úpravu textu existuje několik vlastností. My si ty nejdůležitější ukážeme. Jednu z nich však už známe, tou je vlastnost color, která určuje barvu daného textu.

Práce s textem v CSS funguje velmi podobně, jako úprava textu v aplikaci Microsoft Word.

Zarovnání textu

Text můžeme zarovnat k levému okraji, na střed a nebo k pravému okraji. Vlastnost pro zarovnání textu se jmenuje text-align.

Převedení znaků

Znaky textu můžeme převádět na malá, či velká písmena. Tato vlastnost se jmenuje text-transform.

Textové dekorace

Text můžeme různě nadtrhávat nebo podtrhávat. Tato vlastnost se jmenuje text-decoration.

Velikost textu

Velikost textu nastavujeme díky vlastnosti font-size.

Rodina písem

Každé písmo (font) je jiné. Existují písma patková (serif), bezpatková (sans-serif), strojová (monospace), ručně psaná (script) a spousty dalších. CSS nám umožňuje typ písma nastavit. Rodina písem se nastavuje pomocí vlastnosti font-family.

Tloušťka písma

Díky vlastnosti font-weight můžeme nastavit i tučnost/tloušťku našeho textu.
