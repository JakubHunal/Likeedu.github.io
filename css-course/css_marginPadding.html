Odsazení

Odsazení slouží k tomu, abychom mohli odsadit jeden prvek od druhého. Odsadit můžeme od každé strany (shora, zprava, zdola, zleva).

Máme dva typy odsazení, vnější a vnitřní.

Vnější odsazení

Vnější odsazení se nepočítá do velikosti prvku. Vlastnost pro vnější odsazení se jmenuje margin.

Vnitřní odsazení

Vnitřní odsazení se počítá do velikosti prvku. Vlastnost pro vnitřní odsazení se jmenuje padding.
