Pozicování

Pozicování určuje chování našeho prvku. Určuje kde se prvek vykreslí nebo bude nacházet při skrolování. Vlastnost pro pozicování se jmenuje position. A může obsahovat 4 stranové vlastnosti - top, right, bottom, left. Tyto vlastnosti slouží k tomu, abychom přesně určili pozici, na které se bude náš prvek nacházet.

Statické pozicování

Hodnota pro statické pozicování se jmenuje static. Klasické pozicování, které je v základu nastaveno. Prvek se chová normálně, jak jsme zvyklí. Toto pozicování nemá k dispozici 4, již zmíněné stranové vlastnosti.

Relativní pozicování

Hodnota pro relativní pozicování se jmenuje relative. Toto pozicování je podobné pozicování statickému. Výhodou je, že u tohoto pozicování již můžeme použít 4, již zmíněné stranové vlastnosti.

Absolutní pozicování

Hodnota pro absolutní pozicování se jmenuje absolute. Prvek, kterému nastavíme absolutní pozicování se zachytí/umístí na nejbližší prvek s nastaveným relativním pozicováním. Pokud takový prvek neexistuje, tak se přichytí na tělíčko. Můžeme použít 4, již zmíněné stranové vlastnosti. Toto pozicování je vhodné na dekorace prvků na stránce.

Fixní pozicování

Hodnota pro fixní pozicování se jmenuje fixed. Prvek, kterému nastavíme fixní pozicování se při skrolování hýbe s námi. Říká se, že je tzv. na vyšší vrstvě, než jsou ostatní prvky. Můžeme použít 4, již zmíněné stranové vlastnosti. Toto pozicování je vhodné na fixní navigaci nebo obrázkové pozadí, které poté tvoří zajímavý efekt.

"Lepivé" pozicování

Hodnota pro "lepivé" pozicování se jmenuje sticky. Prvek, kterému nastavíme "lepivé" pozicování se chvíli chová stejně jako relativní prvek, ale pokud se prvek dotkne horní hranice obrazovky při skrolování, tak se přilepí a chová se poté jako prvek s fixním pozicováním. Můžeme použít 4, již zmíněné stranové vlastnosti. Toto pozicování je vhodné na "lepivou" navigaci.
