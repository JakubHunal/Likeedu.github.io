Úvod

CSS jsou kaskádové styly, které slouží pro nastylování/nadesignování naší stránky. Z kurzu o HTML víme, že HTML je kostra naší webové stránky. Tato kostra je však úplně holá, a proto potřebujeme CSS styly, kterými ji ozdobíme.

Vývojové prostředí

Jak HTML, tak i CSS můžeme psát skoro v každém textovém editoru, klidně i v poznámkovém bloku. Ale ne s každým editorem se pěkně pracuje. Já doporučuji si stáhnout editor od společnosti Microsoft - Visual Studio Code, kde je možnost si stáhnout spoustu pluginů (rozšíření, které nám zjednoduší a zpřehlední práci s naším kódem). Odkaz ke stažení VisualStudia Code

Způsoby zápisu CSS

  • 1. Jako element v hlavičce: v hlavičce vytvoříme element s názvem style, do kterého můžeme zapisovat jednotlivé styly
  • 2. Do souboru: vytvoříme soubor s příponou .css a styly do něj poté zapisujeme, tento soubor nalinkujeme do hlavičky
  • 3. Jako atribut tagu: styly můžeme zapsat do atributu style, tento způsob používat nebudeme, neboť je kód poté nečitelný

Všechny typy zapsání stylů si nyní ukážeme.

Jak selektujeme tagy?

Abychom mohli danou část HTML ostylovat, tak ji nejdříve musíme nějak vyselektovat/vybrat. Můžeme selektovat za pomocí HTML tagu, třídy (používáme symbol tečky), id (používáme symbol hashtagu). To, co vyselektujeme se nazývá selektor. Do selektoru poté zapisujeme jednotlivé vlastnosti.

Tagy můžeme i různě skládat za sebe. Podrobněji tím specifikujeme, co za prvek si vybíráme.
