Práce s proměnnými

S různými druhů proměnných se dá pracovat různě. Dvě proměnné se mohou například spojit, porovnat nebo odečíst. Když chceme například sečíst dvě proměnné typu int, tak si vytvoříme novou proměnou typu int s jiným názvem a za znak rovná se napíšeme názvy obou int a mezi ně znak „+“ – tím je sečteme. Příkaz ukončíme samozřejmě středníkem. Pokud tedy byla hodnota jedné proměnné typu int 3 a druhá 5, tak výsledná hodnota nově vytvořeného int bude 8. Int můžeme libovolně kombinovat například odečítat znakem „-“, násobit znakem „*“ nebo dělit znakem „/“. Matematických znaků i proměnných můžeme takto použít samozřejmě více. U proměnné typu double funguje vše stejně jako u typu int, platí vše, co je vysvětleno výše. Viz příklad. 

static void Main(string[] args)
    {
        int i1 = 12;
        int i2 = 4;
        int i3 = i1/i2;
        //hodnota je poté --> i3 = 3
    }

Spojovat se můžeme také proměnné typu string. Můžeme je sčíatt stejně jako proměnné typu int, tedy vytvořit nový string a spojit do něj stávající stringy. Další matematické operace s typem string provádět nemůžeme.

Cvičení

Vytvořte tři stringy se třemi různými slovy, která poté dají dohromady větu. Poté všechna slova spojte do věty vytvořením nové proměnné typu string a tu následně vypište na konzoli.
