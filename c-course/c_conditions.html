Podmínky

Základním typem podmínky je „if“, česky když. Když je splněna nějaká podmínka, tak bude následovat nějaký kód, když podmínka splněna nebude, tak se nestane nic nebo bude následovat také nějaký kód. Základní struktura podmínky vypadá takto: 

static void Main(string[] args)
    {
        if(podmínka){
            //nějaký kód
        }
    }

Podmínek může být také spojeno více pod sebe dohromady za použítí „else if“ – česky nebo když. Blok podmínek vždy začíná jednou podmínkou „if“, poté následuje libovolný počet podmínek „else if“ a vše je zakončeno „else“ – kód, který bude proveden, když nebude splněna ani jedna podmínka. 

static void Main(string[] args)
    {
        if(podmínka) {
            //nějaký kód
        }
        else if(podmínka) {
            //nějaký kód
        }
        else {
            //nějaký kód
        }
 
    }

Podmínka může mít spoustu podob – například pokud se nějaká proměnná rovná „=“ jiné proměnné či definovanému číslo/textu. Dále číselné proměnné můžeme například porovnávat za pomocí znaků větší „>“ a menší „<“.
