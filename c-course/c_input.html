Vstup

V minulé kapitole jsme se podívali na výstupy a výpisy a nyní se podíváme na vstupy. Zkusme si nyní napsat nový kód a do něj jen jeden příkaz „Console.ReadLine();“, který již známe. Poté program spustíme a zkusme si napsat libovolný text či směsici různých znaků. Viz obrázky: 

static void Main(string[] args)
    {
        Console.Readline();
    }

Nyní se vysvětlíme, jak zadaný text načíst do proměnné a poté ji znovu zobrazit. Nejprve si proměnou vytvoříme, a to typu string – jedná se o základní referenční typ, do kterého můžeme uložit jakýkoliv text. Proměnou vytvoříme tak, že napíšeme, co chceme vytvořit, v našem případně tedy „string“, a poté ji pojmenujeme. Dále napíšeme rovná se „=“ a nám již známý příkaz „Console.ReadLine();“, který načte napsaný text. Výsledný příkaz říká, že načtený text se rovná proměnné text, která je typu string.

Nakonec napíšeme námi známý příkaz pro výpis „Console.WriteLine();“ a do závorky napíšeme pouze název proměnné. Aby se program nezavřel tak na konec napíšeme opět příkaz „Console.ReadLine();“. Kód poté spustíme, napíšeme nějaký text a poté stiskneme klávesu „Enter“, tak by se nám měl na druhý řádek vypsat text, který jsme napsali na tom prvním. 

static void Main(string[] args)
    {
        string text = Console.Readline();
        Console.WriteLine(text);
        Console.ReadLine();
    }

Podobným příkazem příkazu „Console.WriteLine();“ je příkaz „Console.Write();“, který na rozdíl od prvně jmenovaného neposune uživatele o řádek níže, ale nechá ho na stejném řádku.

Cvičení

Vytvořte program, který po otevření zobrazí na prvním řádku „Zadejte slovo: “. Uživatel poté napíše za dvojtečku nějaké slovo, a to se mu po stisknutí klávesy Enter vypíše na další řádek.
