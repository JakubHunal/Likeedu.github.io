Výstup

Základy konzolových aplikací jsou vstupy a výstupy (input a output). Vstupy načítají do proměnných zadané hodnoty a výstupy naopak proměnné obrazují. Proměnnou můžeme označit také jako datový typ, který má nějakou hodnotu (data). Později zjistíme, že toto označení je nepřesné, ale zatím si s tím nemusíme lámat hlavu.

Pojďme si zkusit napsat první příklad, který vypíše do konzole větu „Dobrý den uživateli!“. Kurzorem se přesuneme do metody Main (viz obrázek). 

static void Main(string[] args)
    {
        //zde
    }

Začneme psát náš první příkaz, který bude vypadat takto: „Console.WriteLine();“ – viz obrázek. Počáteční slovo „Console“ označuje, že se jedná o příkaz pro konzoli. Slova v programovacím jazyce C# oddělujeme vždy tečkou. „WriteLine()“ je poté název metody pro vypsání konkrétního textu nebo nějaké proměnné. A na konci příkazu je důležité mít středník, tím se ukončuje většina příkazů. 

static void Main(string[] args)
    {
        Console.Writeline();
    }

Nyní máme napsaný první příkaz, ale kdybychom nyní program spustili tlačítkem „Spustit“ v horní liště Visual Studia (ostatní kompilátory mohou mít toto tlačítko jinde), tak se nic nestane – otevře se konzole a hned se zavře. Je to tím, že kompilátor zpracoval celý kód, který mu sice říká, aby něco vypsal, ale neví, co a tak nic nevypíše.

Upřesnění metod se píše do závorek za nimi. Když chceme vypsat předem definovaný text, tak do závorek za metodou vložíme uvozovky a do nich náš text. Po spuštění se ale program opět hned zavře. Je to tím, že program sice náš příkaz provedl, ale hned se poté zavřel.

Aby se konzolový program hned nezavřel, tak na konec kódu (tedy na druhý řádek pod náš příkaz) napíšeme příkaz „Console.ReadLine();“. Tento příkaz zajistí, že se program nezavře hned, ale až po stisknutí klávesy „Enter“ na klávesnici. Když poté program spustíme, tak nám vypíše naši větu „Dobrý den uživateli!“ – viz obrázky. Po stisknutí klávesy „Enter“ se poté zavře. 

static void Main(string[] args)
    {
        Console.Writeline("Dobrý den uživateli!");
        Console.Readline();
    }

Před uvozovky také můžeme napsat takzvané modifikátory – tzn. znaky, které upravují vlastnosti zadaného textu. Například zavináč „@“ nám způsobí, že se text na konci viditelného řádku zalomí a bude pokračovat o řádek níže, v opačném případě by text pokračoval neustále dále doprava po jedno řádku.

Cvičení

Zkuste si napsat nebo zkopírovat nějaký text a ten do uvozovek se zavináčem před nimi vložit. Text by se vám měl po řádcích zalamovat.
