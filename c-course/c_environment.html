Prostředí

V této první kapitole se zaměříme na první prostředí, ve kterém budeme pracovat – tím je konzole (ang. console). Konzolová aplikace je program, který přijímá vstup a zobrazuje výstup na konzoli příkazového řádku. Pro práci s konzolí je nejlepší volbou instalace desktopové aplikace Microsoft Visual Studio, která je zdarma pro Windows i Mac ke stažení z oficiálních stránek. Alternativními možnostmi je použití online C# kompilátorů (kompilátor zpracovává a následně provádí kód) - například .NET Fiddle nebo spoustu dalších C# online kompilátorů.
