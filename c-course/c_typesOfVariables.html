Základní typy proměnných

Do proměnných ukládáme různá data, a právě podle typu dat je rozdělujueme:

  • string (slouží pro uložení libovolného textového pole)
  • int (slouží pro uložení celých čísel)
  • double (slouží pro uložení desetinných čísel s libovolnou délkou číslic za desetinou čárkou)

Proměnné vytváříme tak, že nejdříve deklarujeme, jakou proměnnou chceme vytvořit (typ proměnné) a poté napíšeme její název. Příkaz můžeme ukončit středníkem nebo napsat znak rovná se „=“ a poté u typu string napíšeme do uvozovek hodnotu dané proměnné a u typu int nebo double napíšeme za rovná se rovnou číslo (u double použijeme jako desetinou čárku tečku „.“, nikoliv čárku). Opět, jako všechny příkazy, ukončíme řádek středníkem. 

static void Main(string[] args)
    {
        string s;
        int i;
        double d;
    }
static void Main(string[] args)
    {
        string s = "Ahoj";
        int i = 1;
        double d = 2.548284;
    }

Cvičení

Vytvořte jeden string, int i double a každém předem deklarujte (nastavte) nějakou hodnotu. Hodnoty poté do konzole vypište u každého typu zvlášť na jeden řádek následujícím způsobem: {název proměnné} = {hodnota}.
