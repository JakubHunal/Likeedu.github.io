Třídy a ID

Třídy a ID slouží k zpřehlednění a popsání našeho kódu. Každému tagu, který vytvoříme bychom měli dát nějaký název, abychom vědeli, co to je za prvek a co se v něm asi nachází. Možná si říkáte, že je to zbytečné, když máme tolik tagů, ze kterých si můžeme vybrat, tak proč jim ještě dávat jména. Zbytečnost to není, protože pokud bychom tvořili složitější stránku, tak si jednotlivé prvky musíme nějak pojmenovávat, neboť bychom se v našem kódu za chvíli nevyznali. Třídy a ID jsou důležité, když chceme stránku stylovat/designovat, neboť podle tříd a ID si daný předmět ze stránky vybereme a poté ho ostylujeme/nadesignujeme. Třídy a ID jsou atributy, jako byl například atribut href u odkazu, které zapisujeme k počátečnímu tagu.

Třídy

Třída se značí, dle jejího anglického názvu, tedy class. Stejných tříd můžeme mít na stránce nekonečně mnoho.

ID

ID je unikátní indentifikátor, který se značí id. Může existovat pouze jedno ID se stejným názvem.
