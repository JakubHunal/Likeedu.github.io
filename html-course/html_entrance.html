Úvod

HTML je značkovací jazyk, který slouží k tvorbě webových stránek. HTML tvoří jakousi kostru našich webových stránek. Vše, co na stránce vidíme, každý obrázek, text, nadpis, je vytvořeno právě pomocí HTML zápisu. No a jak vlastně HTML zápis vypadá? Zápis, neboli náš HTML kód je tvořen pomocí určitých značek. Každá značka má jiný význam a používá se trochu na jiných místech na stránce. Všechny důležité značky si v tomto kurzu ukážeme a řekneme si, kde bychom je měli používat. Aby naše webová stránka vypadala hezky, tak se styluje za pomocí tzv. kaskádových stylů CSS. CSS je podrobněji probíráno v kurzu CSS.

Vývojové prostředí

HTML můžeme psát skoro v každém textovém editoru, klidně i v poznámkovém bloku. Ale ne s každým editorem se pěkně pracuje. Já doporučuji si stáhnout editor od společnosti Microsoft - Visual Studio Code, kde je možnost si stáhnout spoustu pluginů (rozšíření, které nám zjednoduší a zpřehlední práci s naším kódem). Odkaz ke stažení VisualStudia Code

Základní pravidla psaní kódu

  • Kód píšeme jedním jazykem
  • Jednotlivé značky se snažíme popisovat, co nejkratším a nejvýstižnějším názvem
  • Nepoužíváme diakritiku, ani speciální znaky
  • Využíváme zápisu CamelCase (např.: moje mama = mojeMama)

Soubor index.html

Soubor index.html je náš hlavní soubor s naším HTML kódem. Tento soubor si můžete pojmenovat jakkoli, ale existuje určité pravidlo, že hlavní webová stránka se jmenuje index a má příponu .html. Pokud bychom tvořili složitější webovou stránku a měli bychom tam více sekcí, neboli více HTML souborů (např.: index.html, kontakt.html, informace.html), tak by se právě jako první zobrazil soubor index.html. Přípona .html nám označuje soubor, do kterého budeme psát náš HTML kód.

Po vytvoření souboru index.html si ho otevřeme. Vidíme, že se nám zobrazila prázdná bílá stránka ve webovém prohlížeči. Nyní přichází na řadu naše vývojové prostředí, ve kterém stránku právě začneme vytvářet. Otevřeme si tedy Visual Studio Code a soubor index tam buď přetáhneme myší nebo zvolíme možnost otevřít soubor nebo složku. Popřípadě můžeme soubor index.html vytvořit rovnou ve Visual Studiu Code. Abychom mohli na stránce vidět úpravy, které jsme do našeho index souboru napsali, tak soubor musíme nejdřív uložit (klávesová zkratka ctrl + s) a prohlížeč obnovit (klávesová zkratka ctrl + r).
