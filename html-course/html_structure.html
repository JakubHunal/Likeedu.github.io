Základní struktura HTML

Základní struktura je něco bez čeho se naše webová stránka neobejde. Každá webová stránka tuto základní strukturu musí nutně mít a nikdo nesmí přeházet posloupnost jednotlivých HTML značek!

V našem index souboru si tedy tuto základní strukturu vytvoříme. Nyní přichází výhoda chytrého vývojové prostředí, jako je Visual Studio Code, neboť stačí zadat příkaz html:5, odentrovat a struktura se sama vytvoří. Všechny části této struktury si vysvětlíme.

Části HTML struktury

  • 1. !DOCTYPE html: značí, že se jedná o HTML5 dokument
  • 2. html: značí, že se jedná o HTML zápis, zahrnujeme sem i jazyk, který stránka používá
  • 3. head: tzv. hlavička, do které se zapisují tzv. meta značky, můžeme do ní importovat fonty, ikonky a spousty dalších
  • 4. meta: touto značkou definujeme metadata (= data o datech) naší stránky, např.: specifikujeme jakou sadu znaků stránka používá, můžeme stránce udělit krátky popisek, uvést autora stránky
  • 5. title: tzv. titulek, hlavní název naší stránky, který se uživatelům zobrazí jako jméno otevřené karty v našem webovém prohlížeči
  • 6. body: tzv. tělíčko, do kterého již tvoříme naši stránku, všechny prvky stránky musí být v této značce
